```uml

@startuml
[*]->トップページ
トップページ : do/商品一覧を表示
トップページ-->商品詳細 : 商品をクリック
  商品詳細 : do/商品の詳細を表示
  商品詳細->購入 :　購入ボタンをクリック
トップページ-down->ログイン :ログインをクリック
  ログイン : do/ログイン認証
  ログイン : entry/id,passを入力
トップページ-->カートの中 :カートをクリック
state カートの中{
  カート -> 購入 : 購入ボタンをクリック
}
state アカウント登録{
[*]-> 登録 
  トップページ-up-> 登録 :アカウントを登録をクリック
  登録 : entry/name,email,id,pass,addressを入力
  登録 ->登録情報確認: アカウント作成をクリック
  登録情報確認 -> 登録完了: はいをクリック
    登録完了: do/アカウント作成
  }
トップページ->検索結果 :検索をクリック
  検索結果 do/検索結果を表示
トップページ-up->プロフィールページ :プロフィールをクリック
state プロフィールページ{
[*] -> プロフィール
プロフィール : do/プロフィールを表示
  プロフィール ->退会ページ :退会するをクリック
  state 退会ページ{
    [*]-> 退会確認 
    退会確認-> 退会しました : はいをクリック
      退会しました : do/アカウント削除
    }
 }
@enduml
```
