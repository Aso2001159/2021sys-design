```uml

@startuml
[*]->トップページ
トップページ : do/商品一覧を表示
トップページ->商品詳細 : 商品をクリック
  商品詳細 : do/商品の詳細を表示
  商品詳細 --> 商品比較結果 : 商品を比較するをクリック
  商品比較結果 : do/比較結果を表示
  商品詳細-> 購入画面 :　購入ボタンをクリック
トップページ-down->ログイン :ログインをクリック
state ログイン{
  state r確認 <<choice>>
  [*] -> ログイン入力 
  ログイン入力 : do/ログイン認証
  ログイン入力 : entry/id,passを入力
  ログイン入力 -> r確認
  r確認 -> トップページ : id&passの一致
   r確認 --> ログインエラー : id or passの不一致
    ログインエラー :do/ログインエラー表示
    ログインエラー : exit/ログイン入力に戻る
  ログインエラー -> ログイン入力 : 再入力

}
トップページ-->カート :カートをクリック
カート --> 購入画面 : 購入ボタンをクリック
state 購入画面{
  [*]-> 購入確認画面
  購入確認画面 : entry/住所,個数を入力
  購入確認画面 -> 購入確定 : 購入を確定するをクリック
  購入確定 : do/商品発注
  購入確定 -> トップページ : トップページに戻るをクリック
}
state アカウント登録{
[*]-> 登録 
  トップページ-up-> アカウント登録 :アカウントを登録をクリック
  state t確認 <<choice>>
  登録 : entry/name,email,id,pass,addressを入力
  登録 ->登録情報確認: アカウント作成をクリック
  登録情報確認 --> t確認 
  t確認 --> 登録エラー : 登録内容不備
    登録エラー : do/エラー表示
    登録エラー : exit/登録に戻る
   登録エラー -> 登録 : 再入力
  t確認 -> 登録完了: 登録内容不備なし&はいをクリック
    登録完了: do/アカウント作成
    登録完了 -> トップページ : トップページに戻るをクリック
  }
トップページ->検索結果 :検索をクリック
  検索結果 : do/検索結果を表示
トップページ-up->プロフィールページ :プロフィールをクリック
state プロフィールページ{
[*] -> プロフィール
プロフィール : do/プロフィールを表示
  プロフィール ->退会ページ :退会するをクリック
  state 退会ページ{
    [*]-> 退会確認 
    退会確認-> 退会しました : はいをクリック
      退会しました : do/アカウント削除
    }
    プロフィール -> プロフィール変更 : プロフィールを変更するをクリック
    state プロフィール変更{
      [*]-> プロフィール変更画面
      プロフィール変更画面 : do/アカウント変更
    }
 }
@enduml

```
