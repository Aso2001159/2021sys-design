@startuml

ログイン -> トップページ : 認証

商品詳細 : entry/個数を選択
商品詳細 -> トップページ : 前の画面へ戻るをクリック

トップページ -> トップページ :ログアウトをクリック

state カート内{
[*] -down-> カート

注文 -> お届け先入力
お届け先入力 -> 入力内容確認 : 入力内容確認をクリック

}
