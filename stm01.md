```uml

@startuml
[*]->トップページ
トップページ : do/商品一覧を表示
トップページ-->商品詳細 : 商品をクリック
  商品詳細->購入 :　購入ボタンをクリック
トップページ-down->ログイン :ログインをクリック
  ログイン : do/ログイン認証
  ログイン : entry/id,passを入力
トップページ-->カートの中 :カートをクリック
トップページ-up->登録 :アカウントを登録をクリック
state アカウント登録{
  登録 : do/アカウント作成
  登録 : entry/name,email,id,pass,addressを入力
  
  }
トップページ->検索結果 :検索をクリック
トップページ-up->プロフィール :プロフィールをクリック
  プロフィール->退会 :退会するをクリック
@enduml
```
